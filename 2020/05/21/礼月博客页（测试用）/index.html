四、块元素

1. 引用

使用 > +空格+引文文本 可以添加引用块。

引用的文本

输入多个 > 可以多级引用

多级引用

2. 无序列表

使用 * 、 + 、 - 都可以创建一个无序列表。

  • 无序列表1
  • 无序列表2
  • 无序列表3

3. 有序列表

使用 1.空格 、 2.空格 、 3.空格 可以创建有序列表。

  1. 有序列表1
  2. 有序列表2
  3. 有序列表3

4. 任务列表[异常]

\ -[ ] 不勾选

\ -[x] 勾选

5. 多行代码块

使用 三个反单引号包围 可以实现多行代码块。

此处为多行

6. 数学表达式[异常]

使用 $$两个美元符包裹 TeX 或 LaTeX 格式的数学公式$$ 可以实现。Typora快捷键 Ctrl+Shift+M

常用公式代码
$$
x^2
$$

7. 插入表格

输入 | 表头1 | 表头2 | 并回车，即可创建一个2x2的列表。Typora快捷键 Ctrl + T 弹出对话框，定义格式

id number
1 2

8. 脚注[异常]

使用 [^1] 可以对文本添加脚注，然后可以去文档最下方编辑脚注内容。

右上为脚注^1

注意：该例子脚注标识是1，脚注标识可以为字母数字下划线，但是暂不支持中文。脚注内容可为任意字符，包括中文。

9. 分割线

输入 *** 或者 --- 再按回车即可绘制一条水平线。

五、文本格式

1. 字体倾斜

使用 *单个星号* 或者 _单下划线_ 可以使字体倾斜。Typora快捷键 Ctrl + I

字体倾斜

2.字体加粗

使用 **两个星号** 或者 __两个下划线__ 可以使字体加粗。Typora快捷键 Ctrl + B

字体加粗

3. 加粗斜体

使用***加粗斜体***可以使字体加粗并倾斜。

加粗斜体

4. 代码标记

代码

5. 删除线

使用~~删除线~~ 可以在将文本上添加一条删除线。Typora快捷键 Alt + Shift + 5

删除线

6. 下划线

使用 <u>下划线的内容</u> 可在文本下添加下划线。Typora快捷键 Ctrl + U

下划线

7. 表情符号[异常]

使用 :单个冒号包围的emoji名: 可以显示emoji表情。

:smile: –​>​ :smile:

8. 下标

使用 <sub>文本</sub>实现下标。

H2O

9. 上标

使用<sup>文本</sup>实现上标。

X2

10.高亮[异常]

使用 ==两个等号== 可以使字体高亮。

==高亮==

11. 文本居中

使用 <center>这是要居中的内容</center> 可以使文本居中。

要居中的文本
